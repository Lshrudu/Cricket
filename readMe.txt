# Algas kalkulators

Īss apraksts: Python programma, kas aprēķina neto algu no bruto algas, ņemot vērā VSAOI, IIN, neapliekamo minimumu un atvieglojumu par apgādājamiem. Paredzēta mācību nolūkiem.

## Funkcionalitāte
- [x] Bruto algas ievade un validēšana
- [x] VSAOI aprēķins
- [x] IIN aprēķins
- [x] Neto algas aprēķins

## Prasības
- Python 3.10+
- Papildu bibliotēkas nav nepieciešamas

1. Lejupielādē projektu
2. Atver mapi terminālī

## Palaišana
python algas_kalkulators.py

## Lietošana (īss piemērs)
Ievadi: 1000  
Ievadi apgādājamo skaitu: 1  

Izvadi:
